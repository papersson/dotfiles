# Minimal Neovim Configuration

A lightweight, navigation-focused Neovim setup for mastering core Vim capabilities.

## Philosophy
- No LSP - Navigate with tags and grep
- No AI - Focus on understanding code
- Core Vim - Master fundamentals that work everywhere

## Quick Start
1. Open Neovim: `nvim`
2. Install plugins: `:Lazy sync`
3. Generate tags: `ctags -R .` (in your project)

## Key Features
- **Telescope** - Fuzzy finding and search
- **Treesitter** - Syntax-aware navigation
- **Oil.nvim** - File explorer as buffers
- **Flash.nvim** - Enhanced motions
- **Vim-fugitive** - Git integration
